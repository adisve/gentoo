# Overview

These are my Gentoo dotfiles. The following packages are required in order to properly use the supplied config files:

- [i3](https://packages.gentoo.org/packages/x11-wm/i3)
- [picom](https://packages.gentoo.org/packages/x11-misc/picom)
- [polybar](https://packages.gentoo.org/packages/x11-misc/polybar)
- [zsh](https://packages.gentoo.org/packages/app-shells/zsh)
- [vim](https://packages.gentoo.org/packages/app-editors/vim)
- [pulseaudio](https://packages.gentoo.org/packages/media-sound/pulseaudio)
- [neofetch](https://packages.gentoo.org/packages/app-misc/neofetch)
- [alacritty](https://packages.gentoo.org/packages/x11-terms/alacritty)
- [dconf](https://packages.gentoo.org/packages/gnome-base/dconf)
- [starship](https://starship.rs/)
